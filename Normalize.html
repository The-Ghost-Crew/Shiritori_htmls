Additional normalizations
Add extra accepted variants for a specific word. The standard normalization always runs — this adds on top of it.
Example: 007 → double-o-seven will show alongside the auto-generated zero-zero-seven.

All output must only contain a–z, - (hyphen), and ' (apostrophe). Words cannot start or end with a hyphen.


Title abbreviations — absorb a following period and space:

drmsmr mrsstrd mt

Punctuation quick reference:

/ \ → hyphen (merges with adjacent -)  ·  ~ _ → hyphen (no merge, ignored at start/end)  ·  . → hyphen (any count = 1, ignored at start/end)  ·  , → hyphen (any count = 1, ignored at start/end)  ·  : → hyphen (:: = colon-colon, ::: = colon-colon-colon)  ·  ; → hyphen (;; = semicolon/semi-colon variants)  ·  + → plus segment (each + is its own plus)  ·  () [] {} → paired = hyphen, unpaired = parenthesis/bracket/curly-bracket  ·  ! → exclamation-mark / exclamation-point / omit  ·  ? → question-mark / omit  ·  apostrophe before a generated hyphen → hyphen removed
